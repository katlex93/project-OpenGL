# OpenGL Project (LearnOpenGL)

## Build Instructions

### 1. Configure (run once or after CMakeLists.txt changes)
```bash
cmake ..
```

### 2. Build (run after every .cpp/.h change) . From the build folder
```
cmake --build .
```

### 3. Run. From the build folder
```
./OpenGLProject
```
### 4. Link to project 
```
./OpenGLProject
```
